# Conway's Game of Life - Java & SQL Server

## 📌 Project Overview
A robust implementation of Conway's cellular automaton, focusing on Software Engineering best practices and data persistence.

## 🛠 Tech Stack
* **Language:** Java
* **Database:** MS SQL Server (T-SQL)
* **Testing:** JUnit 5
* **Patterns:** Singleton, Factory

## 🚀 Key Features
* **State Persistence:** Saves and loads grid configurations using SQL Server.
* **Unit Testing:** Validates all 4 core transition rules with JUnit.
* **Clean Architecture:** Modular design using standard OOP patterns.
